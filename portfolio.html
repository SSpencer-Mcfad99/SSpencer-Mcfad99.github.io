Review Site (Assignment)

If you wish just to skip down to links related to this site, click the button below!


Project Info

The review site was something that I did for my front-end web development module. The concept behind the site was for a user to be able to upload reviews about a game they played with the game name, a star rating and a description of how they found the game. Considering we could only use pure html/CSS/JS, there was no back-end or framework involvement for this assignment.

Problems Faced

The biggest problems I had during development was definitely formatting the site to fit in without overflowing elements. A huge example is the rating stars in the createreviews page. There were issues with the selected stars appear after the unlit stars rather than replacing it. There were also issues with ensuring that the stars fit in the form on smaller devices. If you want more details about my experiences during the assignment, you can view the readme file in the repo (link to my Github can be accessed in the aboutme page where you can find said repo).
