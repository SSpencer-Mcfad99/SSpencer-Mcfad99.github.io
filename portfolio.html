Review Site (Assignment)

If you wish just to skip down to links related to this site, click the button below!


Project Info

The review site was something that I did for my front-end web development module. The concept behind the site was for a user to be able to upload reviews about a game they played with the game name, a star rating and a description of how they found the game. Considering we could only use pure html/CSS/JS, there was no back-end or framework involvement for this assignment.

Versions

Whilst the functionality of the site was really good, the appearance of the site is not that good. So after completing the assignment, I decided to do a version of the site where the appearance looks ok rather than really bad. Voting System Site (Assignment)

Project Info

The concept behind this project was to create a site where people could learn about different voting systems and how voting works. This was a project where I had to do both front and back end web development of the site (this ultimately came at a cost for the appearance of the site). For this project, I also had to learn how to set up a web server to host the website on due to the situation partway through my final year. It was a mixture of this and the cost of hosting sites and Domain Name services is the reason why this site is not active (although it's still in a repo but you can't view the site itself).
